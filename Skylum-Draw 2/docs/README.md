# Skylum-Darw 2

Aplikasi menggambar berbasis web.

## Fitur

- Kanvas gambar interaktif
- Pilihan warna dan ukuran kuas
- Undo/redo
- Simpan hasil gambar

## Instalasi

1. Clone repositori:
   ```
   git clone <url-repo>
   ```
2. Masuk ke folder proyek:
   ```
   cd Skylum-Darw 2
   ```
3. Install dependensi:
   ```
   npm install
   ```
4. Jalankan aplikasi:
   ```
   npm start
   ```
